# 项目名称：排序算法可视化工具
用Python实现多个排序算法 + 动态可视化过程 + 性能对比
# 项目核心目标
- 实现至少 3 种排序算法（如：冒泡、选择、插入、归并、快排）
- 用 matplotlib 动画展示排序过程（柱状图高度变化）
- 支持选择不同算法、不同数据规模、不同初始顺序（随机/逆序/已排序） 
--（进阶）记录并对比各算法运行时间，生成性能报告
